# Tic-Tac-Toe Game 🎮

A simple console-based Tic-Tac-Toe game developed using C++. This project allows two players to play Tic-Tac-Toe in the terminal while demonstrating fundamental C++ programming concepts such as arrays, loops, functions, and conditional statements.

## 📌 Features

- Two-player gameplay (X vs O)
- Interactive console interface
- Win detection for rows, columns, and diagonals
- Draw detection
- Input validation to prevent invalid moves
- Easy-to-understand code structure

## 🛠️ Technologies Used

- C++
- Standard Template Library (STL)
- Console/Terminal

##Project Structure 
🎯 Game Rules
The game is played on a 3×3 grid.
Player X goes first, followed by Player O.
Players take turns choosing positions from 1 to 9.
The first player to align three marks horizontally, vertically, or diagonally wins.
If all positions are filled and no player wins, the game ends in a draw.
